HTML/CSS Reference

HTML

HTML Tag Name Description Example
<html> HTML The <html> tag represents the root of an HTML document.
The <html> is the container for all other HTML elements
 <!DOCTYPE html>
<html>
Insert code here. Include the head and body tags
</html>
<head> Head The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

This tag contains links (fonts or stylesheets), meta data, and title data.
 <head>
Insert any links to CSS stylesheets, title of your webpage and meta data
</head>
<body> Body The <body> tag defines the document's body.
The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
 <body>
Enter your heading
Enter a paragraph
Enter links, images, videos, or gifs
</body>
<h1> - <h6> Heading(s) The <h1> - <h6> tags are used to degine HTML headings.
<h1> degines the most important heading. <h6> defines the least important heading

NOTE: only use one <h1> per page. This should represent the main heading/subject for the whole page
 <body>
<h1-h6> "This is a heading" <h1-h6>
</body>
<img> Image The <img> tag is used to embed an image in an HTML page.
Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.

The <img> tag has two required attributes:
- src - Specifies the path to the image
- alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
 <img src="image.jpg" alt="image" width="" height="">
<video> Video The HTML <video> element is used to show a video on a web page. <video width="" height="" controls >

<source src ="movie.mp4" type ="video/mp4">
This browser doesn't support video tag.

</video>
<audio> Audio To play an audio file in HTML, use the <audio> element.

You can also add controls and/or autoplay
 <audio controls>

<source src="audio.mp3" type="audio/mp3">
Your browser does not support the audio element.

</audio>
<a> Hyperlink The <a> tag defines a hyperlink, which is used to link from one page to another.

The most important attribute of the <a> element is the href attribute, which indicates the link's destination. 		<a href="website.com">This is a hyper link. </a>
<header> Header The <header> element represents a container for introductory content or a set of navigational links.

A <header> element typically contains:

-one or more heading elements (<h1> - <h6>)
-logo or icon
-authorship information 		<header>

<h1> Heading here </h1>
<p> Additional info </p>

</header>
<footer> Footer The <footer> tag defines a footer for a document or section.

A <footer> element typically contains:

-authorship information
-copyright information
-contact information
-back to top links
-Etc 		<footer>

<p> Author | Links | Top of page | Contact

</footer>

CSS

Name Description Example
Syntax Selector {

Property: Decleration;

} 		h1 {

color: Red;
text-align: center;
}
Hexadecimal Colors A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. Selector {

color: #8FBC8F;
}

This color is (darkseagreen).
RGB Colors An RGB color value represents RED, GREEN, and BLUE light sources.

In CSS, a color can be specified as an RGB value, using this formula:
rgb(red, green, blue)

Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255. 		Selector {

color: rgb(xxx,xxx,xxx);
}

Input values from 0 to 255.
Hue, Saturation, and Lightness (HSL) Colors HSL stands for hue, saturation, and lightness.

HSL color values are specified with: hsl(hue, saturation, lightness).

Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, 240 is blue.

Saturation is a percentage value; 0% means a shade of gray and 100% is the full color.

Lightness is also a percentage; 0% is black, 100% is white. 		Selector {

color: hsl(180,50%,50%);
}
Opacity and Alpha Alpha is a decimal number from zero to one.
If alpha is zero, the color will be completely transparent. If alpha is one, the color will be opaque.
The value for half-transparent would be 0.5.
 Selector {

color: hsla(180,50%,50%,0.5);
}

Selector {

color: rgba(xxx,xxx,xxx,0.5);
}
Font-Family The font-family property specifies the font for an element.

The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font. 		Selector {

font-family: "Times New Roman";
}
Fall Back Fonts CSS font Fallbacks provide a backup for the fonts i.e. if one font doesn't work properly then the browser will try the other one.

For good coding practice write a generic font family at the end of the list and choose the font fallback within the same font family. 		Selector {

font-family: Carlson, Georgia, "Times New Roman";
}
Serif and Sans-Serif Serif fonts have extra details on the ends of each letter

Opposed to Sans-Serif fonts, which do not have the extra details.

Serif and sans-serif are also keyword values that can be added as a final fallback font if nothing else in the font stack is available. 		"Times New Roman" would be considered a serif font.

"Calibri" would be considerd a sans-serif font.
Font Weight In CSS, the font-weight property controls how bold or thin text appears.

It can be specified with keywords or numerical values. 		Selector {

font-weight: bold;
}
Font Style The font-style property specifies the font style for a text.

Fonts can be:
normal, italic, oblique, initial, inherit
 Selector {

font-style: italic;
}
Google Web Fonts Fonts can also be added using a @font-face ruleset in your CSS stylesheet instead of using a link element in your HTML document.

You can download a specific font from Google fonts to obtain the file(s) and it can be stored in your project folder.
Once inside your folder for projects, you can use @font-face property in CSS to style the HTML document with the downloaded font

It is important to note that you should use fallback file formats such WOFF, WOFF 2 and truetype 		Selector {

font-family: 'MyFont'; src:
url('fonts/Roboto.woff2') format('woff2')
url('fonts/Roboto.woff2') format('woff')
url('fonts/Roboto.woff2') format('truetype')

}
